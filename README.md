
# Stripchat Recorder

Android application supports recording stripchat


## How to use
You just need to get the stripchat link by copying the url on the browser

## Features

- Support hls videos
- Download tiktok videos
- Save history
- Play/Record Livestream Tiktok
- Play/Record Stripchat
